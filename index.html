Meet Lisp

Lisp is a special language among the giants (Java, C, Python) in that it’s a functional language: everything in the language is done by a call to a function. For example, (+ 2 3) is how you would add 2 and 3. Of course you can define a function, and this itself involves a call to the function defun (defun myfunction (v1 v2) (* v1 v2)) The code above defines a function that takes two variables as input and returns the product of them.

Programming: what is passing?

In Python, so-called primitive types are passed by value. >>> x = 5 >>> x 5 >>> y = x >>> y 5 >>> y = 7 >>> y 7 >>> x 5 Other types are passed by reference. This means that the memory location of an object is passed, and modifications (even to a different variable name that’s pointing to the same location) affect that one object.
